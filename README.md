# ⌨️ Speed Typing Test

A modern Speed Typing Test web application that helps users improve their typing speed and accuracy.

## 🚀 Live Demo

https://soorajstudio.github.io/Speed-Typing/

---

## 📌 Features

- Real-time typing speed calculation
- Words Per Minute (WPM)
- Accuracy calculation
- Timer
- Mistake counter
- Performance Dashboard
- Responsive UI

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
